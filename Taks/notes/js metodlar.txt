document.getElementById(id) -return 
 yazdiğımız kodda id funksiyası olan elementi qaytarır.
yəni id verdiyimiz elementi idarə etmək üçün metoddur.əgər bu id aid hec bir element və ya kod
yoxdusa o bize "null" qaytarır. null o deməkdirki yəni onun içində belə element yoxdu.
eyni id ilə çoxlu kod varsa bu metodda sadece ən birinci yazılanı icra eliyir.

document.getElementsByTagName(name)-return
yazığımız koddu bize html collection obyekti olaraq qaytarır.html -collection dediyimizdə odur ki array siyahısıdır.
array- bir çox element bir yere yığmaq üçün metoddur.
bu metodda elemente indeks verdikde 0 dan başlayaraq bütün elementləri qaytarır.

document.getElementsByClassName(name)-return
kodda olan bütün elementləri verdiyimiz class adina görə HTMLCOLLection object kimi qaytara bilir.
object-massiv yaratmağa imkan verir.obyekt bir ad verib onun içərisində dəyərlər verib istifadə edə bilərik.
əgər metodun sonunda nöqtə qoya biliremsə objectdi.

document.querySelector()-return
bu metod css fayılında yazdığımız kodlarda uyğun elementləri qaytarır.
bu metod uyğun olan kodlardan ilk elementi qaytarır.

onda uyğun olan element ilk elementi qaytarır.
